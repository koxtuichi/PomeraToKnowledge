# ポメラ → ブログ 送信テンプレート

ポメラからメールでブログ草案を送信する際のフォーマットです。
件名を「BLOG」にし、本文を以下のテンプレートに沿って書いてください。

---

## テンプレート

```
テーマ: ここに物語のテーマを書く
ジャンル: 日常 / 仕事 / 子育て / テクノロジー / 人間関係
トーン: ほっこり / シリアス / コミカル / 哲学的

伝えたいこと:
・読者に伝えたいメッセージ1
・読者に伝えたいメッセージ2

エピソード:
物語の種になる体験やエピソードを自由に書く。
箇条書きでも文章でもOK。
AIがこれをフィクションの物語に変換します。

読後感: 読み終わった人にどんな気持ちになってほしいか
```

---

## 記入例

```
テーマ: 小さなツールが生活を変える
ジャンル: テクノロジー
トーン: ほっこり

伝えたいこと:
・機能が少ないことが強みになることがある
・制限が集中力を生む

エピソード:
ネットに繋がらない文章入力専用機を使い始めた。
最初は不便だと思ったが、通知が来ないので
2時間で書ける量が倍になった。
画面はモノクロで余計な誘惑がない。

読後感: 何か一つ、引き算してみようかなと思える
```

---

## 注意点

- **テンプレートなしでも動きます** — 自由形式で書いた場合、AIが全文をエピソードとして解釈します
- **「エピソード」は着想の素材です** — AIが架空の登場人物と舞台でフィクション短編に仕上げます。個人情報はブログに出ません
- **トーンは大事です** — ここを指定すると出力の雰囲気が安定します
