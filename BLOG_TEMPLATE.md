# ポメラ → ブログ記事 送信テンプレート

ポメラからメールでブログのネタを送信する際のフォーマットです。
件名を「BLOG」にし、本文にブログにしたいメモを書いてください。

---

## テンプレート

```
テーマ: ここにブログのテーマを書く
カテゴリ: テクノロジー / 子育て / 日常 / 仕事 / ライフハック

伝えたいこと:
・読者に伝えたいメッセージ1
・読者に伝えたいメッセージ2

メモ:
ブログに書きたいことを自由に書く。
箇条書きでも文章でもOK。
AIがこれを読みやすいブログ記事に仕上げます。
```

---

## 記入例

```
テーマ: ポメラが最高のアウトプットツールである理由
カテゴリ: テクノロジー

伝えたいこと:
・機能が少ないことが強みになることがある
・制限が集中力を生む
・書くことだけに集中できる環境の大切さ

メモ:
ネットに繋がらない文章入力専用機を使い始めた。
最初は不便だと思ったが、通知が来ないので
2時間で書ける量が倍になった。
画面はモノクロで余計な誘惑がない。
PCだと調べ物を始めて脱線してしまうが、
ポメラだと書くしかないので集中できる。
```

---

## 注意点

- **テンプレートなしでも動きます** — 自由形式で書いた場合、AIが全文をメモとして解釈して記事にします
- **個人情報は保護されます** — AIが記事に仕上げる際、固有名詞は必要に応じて一般化されます
- **ナレッジグラフが活用されます** — 日記から蓄積された知見や関心事が記事の深みに反映されます
